# Flappy Bird Clone

## About
This is a clone I made of Flappy Bird to learn game development in Godot.

## Gameplay
Simply click the left mouse button and keep moving without colliding with pipes to get the highest score possible. If you fall, you can use the restart button to play again.

[Watch gameplay on YouTube](https://youtube.com/shorts/7KV7ubEFvPU?feature=share)

## Technologies Used
- Godot
- GDScript

## How to Run
Install the version for your operating system from the Releases section.
